Estrutura mínima do html


<!doctype html> <!-- HTML5 doctype -->
<html lang='pt-br'>
    <head>
        <meta http-equiv="X-UA-Compatible" content="IE=Edge" /> <!-- Impede que o I.E. use o modo de compatibilidade -->
        <meta charset="UTF-8"> <!-- forçar o encoding UTF-8 para funcionar corretamente a acentuação -->
	    <title>Example Page</title>
        <script src="./script/rib-controles-min.js"></script>
        <link href="./css/style.css"rel="stylesheet" />
        <link href="./fontawesome-5.15.3/css/all.css" rel="stylesheet"> <!-- Para incorporar ícones -->
    </head>
    <body>
        <header id="ID_DO_HEADER_DA_PAGINA"></header> <!-- bloco onde será encapsulado o controle $rib.header()-->
        <div id="ID_DO_PAINEL DE PESQUISA"></div> <!-- bloco opcional de pesquisa -->

        <div id="conteudo"> 
            <!-- BLOCO ONDE O CONTEÚDO DA PAGINA SERÁ COLOCADO (OBRIGATÓRIO O ID="CONTEUDO")-->
        </div>
                     
        <div id="rib-loading" class="fundo"> <!-- exibir a tela de loading ao carregar a pagina -->
            <span class="ico-spinner"></span>
        </div>
    </body>
</html>

.button

O estilo para botões de ação podem ser aplicados aos seguintes objetos html: <a>, <div>, <span>, <input>, <button>

EXEMPLOS:


    <div class="button primary" onclick="yesSir();">Ok</div>
    <a class="button alert" href="#" onclick="yesSir();">Ok</a>
    <span class="button warning" onclick="yesSir();">Ok</span>
    <button type="button" onclick="yesSir();">Ok</button>
    <input type="button" onclick="yesSir();">Ok</input>

.bloco

Insere no conteudo da página um painel que pode ser de cor cinza ou branco conforme exemplo abaixo:


    <!-- Estrutura mínima do bloco -->
    <div class="bloco"> 
        <div class="header g-arrow-dn">Bloco normal</div>  <!-- header do bloco é opcional e sub-classe 'g-arrow-dn' também-->
        <div class="conteudo">
            Conteudo do bloco cinza
        </div>
    </div>
    <div class="bloco branco"> <!-- para o bloco basta adicionar a sub-classe "branco" à classe bloco -->
        <div class="conteudo" style="text-align:center;" >
            <p>Conteudo do bloco (cinza)</p>
            <input type="button" value="Ok" />
        </div>
    </div>

.nowrap

Usado para que um <label> mantenha-se sempre próximo ao controle (<input>) ao redimensionar a tela:

Redimensione a pagina para identificar o efeito


    <div class="bloco">
        <div class="conteudo">
            <div>
                <span class="nowrap"><label>Nome:</label> <input type="text" /></span>
                <span class="nowrap"><label>Documento:</label> <input type="text" /></span>
                <span class="nowrap"><label>Endereço:</label> <input type="text" /></span>
                <span class="nowrap"><label>CEP:</label> <input type="text" /></span>
            </div>
            <div>
                <span class="nowrap top"><label>Nome:</label> <input type="text" /></span>
                <span class="nowrap top"><label>Documento:</label> <input type="text" /></span>
                <span class="nowrap top"><label>Endereço:</label> <input type="text" /></span>

Componentes Rib

Header

Assinatura: $rib.header( id, params)

Parametros: id (ID do objeto)

params (JSON object)


        {
            title        : string
            user         : string //User Name logged in
            subtitle     : string //Subtitle
            logoURL      : string //Company Logo URL
        }

Metodos: addMenuOption(opts) (opts = JSON object)

EXEMPLO (este exemplo está sendo usado no header desta pagina)


$rib.ready(function()
{
    //Objeto com todos os itens do menu a serem adicionados (e que podem ser trazidos do code-behind)
    var menu = {
                'style-deflt': { action: 'changeCss();'       , icon: 'fas fa-buy-n-large' },
                'style-blue' : { action: 'changeCss("blue");' , icon: 'fas fa-old-republic' },
                'style-red'  : { action: 'changeCss("red");'  , icon: 'fas fa-jedi' },
                'style-green': { action: 'changeCss("green");' },
                'style-gold' : { action: 'changeCss("gold");' },
                'Aprovar'    : { options: { 'Conforme proposta': {

                        options: {  'Opt 1': { icon: 'fas fa-trophy'    },
                                                    'Opt 2': { icon: 'fas fa-star-half-alt' },
                                                    'Opt 3': { icon: 'far fa-star' }	}
                                },
                                            'Com alterações'   : { icon: 'fas fa-user-astronaut' },
                                            'Com restrições'   : { icon: 'fas fa-buy-n-large' } 
                                        }
                            },
                'Provisionar/Postergar Pagamento de Boleto': { action: 'changeStatus("disabled");' }
            };

    //Instancia o RibHeader e em seguida adiciona a lista de menus
    $rib.header("header-rib",{ title: 'Application Title', user : 'Ribeiro, Elber' }).addMenuOption(menu);
}

Chamadas AJAX

new RibPOST(url, parametros) Retorna uma promise com o resultado (then) ou falha (catch)

new RibGET(url, parametros) Retorna uma promise com o resultado (then) ou falha (catch)

new RibDownload(parametros) Retorna um arquivo

Nome da pagina que estamos chamando (ex. 'page.aspx').

(opcional) Informar a lista de parametros do método que estamos chamando (no formato JSON)

default => true. Exibir ou não a animação de loading

Caso showLoading = true, exibe uma mensagem em conjunto.

default => true. Não usar false!!!!! (DEPRECATED)

default => 'application/json; charset=utf-8'

default => 'json'. Informar o tipo de retorno
